# Tennis Club Website — Project Rules

## Stack
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Language: TypeScript

## Design
- Always reference Figma for layout, spacing, and components
- Primary font: Segoe UI (headings and UI elements)
- Secondary font: Roboto (body text)
- Font weights in use: light, regular, medium, bold
- Never invent colors or spacing — pull from Figma

## Rules
- Mobile responsive by default
- Keep components small and reusable
- Use descriptive file names (e.g. HeroSection.tsx, BookingCard.tsx)
- Never hardcode colors — use CSS variables or Tailwind config
